Projects

E-commerce Website

I have designed a fully functional e-commerce website where users can browse and purchase items for themselves. The frontend is built using React.js with JSX for component structure and CSS for styling, ensuring a responsive and user-friendly interface. JWT authentication is implemented to securely manage user login and sessions. The platform also utilizes JWT for storing cart product data, allowing users to maintain their cart across sessions. On the backend, Node.js and Express.js handle API requests, ensuring smooth data flow and operations. The database is powered by MongoDB, efficiently storing user data, product details, and order history.

Ticket Management System

I have designed and developed a full-stack Ticket Management System to streamline issue tracking and resolution. The backend is built using C# and the .NET Framework, providing a robust and scalable architecture for handling ticket operations. A responsive and interactive user interface was created using HTML, CSS, and JavaScript, ensuring a seamless user experience. SQL Server is integrated to efficiently store and manage ticket data, with optimized queries for fast retrieval. The system includes features for ticket creation, assignment, status tracking, and resolution history, enhancing workflow efficiency. This solution improves issue resolution processes and enhances overall productivity.

VBR Fabrication

I have designed and developed a dynamic website for VBR Fabrication, a company specializing in complete truck manufacturing. The website features dedicated sections for Home, About, Services, Projects, and Contact Us, providing comprehensive company information. I developed both the frontend and backend, ensuring a well-structured and efficient system for content management. A SQL database was implemented using MySQL to store and manage data securely and efficiently. The website is optimized for responsiveness and seamless navigation, enhancing user experience across devices. Bootstrap, along with HTML, CSS, JavaScript, and PHP, was used to ensure a visually appealing and functional design.
